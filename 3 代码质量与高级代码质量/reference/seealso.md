# \<seealso\>

## 语法 Syntax

``` csharp
<seealso cref="member"/>  
```  

## 参数 Parameters
* cref = " member"
 对可从当前编译环境调用的成员或字段的引用。 编译器检查是否存在给定的码位元素，并将 member 传递到输出 XML 中的元素名称。member 必须在双引号 (" ") 内。

## 备注 Remarks
使用 \<seealso\> 标记，可以指定想要在“另请参阅”部分中显示的文本。 使用 [\<see\>](see.md) 从文本内指定链接。
